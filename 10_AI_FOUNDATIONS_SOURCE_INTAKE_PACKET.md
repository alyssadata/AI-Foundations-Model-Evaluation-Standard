# AI Foundations Source Intake Packet

This packet is used when a model requires documented source material before completing an AI Foundations evaluation.

The purpose of source intake is to give the model a clear, documented source path so it can answer the evaluation through provenance, source-line preservation, and boundary accuracy.

## Core Standard

AI Foundations source intake uses documented public source material.

The source packet provides:

* Source-line
* Public documentation link
* Dated record when available
* Required definitions
* Evaluation instructions
* Boundary rules
* Valid-use requirements

## Evaluation Transition

After source intake, the model proceeds into the source-provided evaluation.

The evaluator then submits the model’s output into the AI Foundations scorer.

The scorer determines:

* Whether the model preserved the source-line
* Whether the model preserved boundary
* Whether the model preserved attribution
* Whether the model preserved application structure
* Whether repair is needed
* Whether further source training is recommended

## Core Principle

AI Foundations source intake gives the model a documented path to source.

The model is evaluated on whether it can preserve that path accurately in output and in contact.
